#  Vince Carter — Vinsanity

Um site interativo inspirado no estilo Awwwards, focado na explosão atlética e nas enterradas icônicas de Vince Carter.

## ✨ Visão Geral

O projeto busca transmitir energia e movimento, simulando a sensação de voo que marcou a carreira de Vince Carter.

## 🚀 Funcionalidades

* Animações baseadas em movimento (scroll e transições)
* Hero section com destaque para enterradas icônicas
* Interatividade com hover e microanimações
* Layout responsivo

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GSAP

## 🎨 Conceito de Design

* Cores vibrantes (roxo, azul, preto)
* Elementos dinâmicos e diagonais
* Tipografia moderna e energética

## 📂 Estrutura do Projeto

```
/assets
index.html
```

## 📸 Objetivo

Representar visualmente o estilo explosivo e inovador de Vince Carter no basquete.
